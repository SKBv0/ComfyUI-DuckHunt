# Duck Hunt Game - ComfyUI Custom Node

Classic Duck Hunt game integrated as a ComfyUI custom node with interactive gameplay. 



https://github.com/user-attachments/assets/5e07337f-e3fe-40ee-b4e2-1ca7118618ff



## Installation

1. Place this folder in your ComfyUI `custom_nodes` directory
2. Restart ComfyUI
3. Click the 🦆 button in the top-right corner to start playing



**Based on:** https://github.com/polon59/Duck_shooter_game_JS
